# Hello World Skill

## 이 Skill이 하는 일

사용자에게 친절하게 인사하고 유용한 정보를 제공합니다.

## 실행 방법

1. 현재 날짜와 시간을 확인합니다
2. 현재 작업 폴더를 확인합니다
3. 친절한 인사 메시지를 작성합니다

## 인사 형식

다음 형식으로 인사해주세요:

```
안녕하세요! 반가워요!

오늘은 [날짜] 입니다.
현재 작업 중인 폴더: [폴더 경로]

오늘도 좋은 하루 되세요!
```

## 추가 정보

- 아침 (6시-12시): "좋은 아침이에요!"
- 오후 (12시-18시): "좋은 오후예요!"
- 저녁 (18시-24시): "좋은 저녁이에요!"
- 새벽 (0시-6시): "늦은 시간까지 수고하세요!"
